# Forge Overview

Hammer Forge Studio is a simplified GUI for generating ML pipeline i.e No code environment. We connect different nodes to make an ML pipeline. Studio containerize an ML pipeline that can execute with some guarantees with any ML framework, on any device/sensor, and any chipset.
